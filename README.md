## WorkTool

本安卓应用是一个依附于企业微信来运行的无人值守群管理机器人，非hook侵入式函数修改，使用安卓系统原生支持的无障碍服务能力，手机无需Root，应用兼容99%的手机，长时间运行稳定且因为没有修改系统和软件所以理论上不存在被外挂监测和封号风险。
